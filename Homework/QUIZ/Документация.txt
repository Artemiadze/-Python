Файлы:
1. quiz.py - основной файл с кодом 
2. Interaction_With_Question.py - файл, содержащий все функции для работы с викторинами
3. Question.txt - текстовый файл где хранятся тестовые вопросы (используется по дефолту)
4.:
     4.1 Answer_Options.txt - текстовый файл, где хранятся все варианты ответов для вопросов из файла Question.txt (в этом файле 4 варианта ответов на каждый вопрос) (используется по дефолту)
     4.2 test.txt - текстовый файл для доп тестирования моей программы, где хранятся все варианты ответов для вопросов из файла Question.txt (в этом файле на каждый вопрос из файла Question.txt приходится уже разное кол-во вариантов ответов, например, на первый вопрос 3 варианта ответов, на второй уже 5 вариантов).
5.:
     5.1 Correct_Answers.txt - текстовый файл, где хранятся правильные ответы для файла Answer_Options.txt, где в одной строке 1 одна цифра - номер варианта (используется по дефолту)
     5.2 test_Correct.txt - текстовый файл для доп тестирования моей программы, где хранятся правильные ответы для файла test.txt, где в одной строке 1 одна цифра- номер варианта
6. Points.txt -  текстовый файл, в котором лежат баллы за каждое задание (используется по дефолту)

Работа с тестами:
В начале файла  quiz.py программа открывает 4 файла: файл с вопросами, вариантами ответов, правильными ответамии и баллами за каждое задание. Если хотите в тестировании использовать тот тест, 
где вместо 4 варианта ответов (файл Answer_Options.txt) на каждый вопрос будет произвольно задано кол-во вариантов ответов (test.txt), то нужно просто в начале файла  quiz.py поменять Answer_Options.txt
на  test.txt (тоже самое еще в 18 строке кода) и обязательно Correct_Answers.txt на test_Correct.txt.
